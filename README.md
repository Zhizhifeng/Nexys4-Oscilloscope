# Nexys4-Oscilloscope

这是华中科技大学电子信息与通信学院《模拟电路与数字系统（三）实验》课程的结课项目，由两人完成。

本项目基于Xilinx Artix 7 FPGA上集成的XADC模块实现1MSPS的双通道模拟信号采样，利用Verilog设计硬件电路对采样数据进行触发检测与测量，并将数据最终存储到BRAM中。项目还借助Nexys 4开发板的外围I/O设备进行示波器处理参数的控制。

最终项目存储在test_1文件夹中，使用的工具为Vivado 2022.2。

## 功能

1. 通过XADC实现双通道采样
2. 通过开关和按键实现下列参数的调整
  - 触发电平
  - 垂直刻度
  - 水平刻度
  - 触发通道
3. 各通道最大值、最小值的测量
4. 通过VGA在屏幕上显示下列内容
  - 背景网格
  - 波形图像（x-t模式与x-y模式）
  - 各通道的测量值
  - 触发设置
